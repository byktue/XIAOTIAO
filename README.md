激活conda环境
```bash
conda activate xiaotiao1
```

下载nodejs依赖
```bash
conda install -c conda-forge nodejs -y
```

查看下载结果
```bash
node -v
npm -v
```

下载相关环境依赖
```bash
cd uni-preset-vue-vite
```

```bash
npm install
```

位于uni-preset-vue-vite目录下
启动dev server
```bash
npm run dev:h5
```

用浏览器 DevTools 的设备工具栏（Ctrl+Shift+M），配合常见视口尺寸即可快速预览手机页面
